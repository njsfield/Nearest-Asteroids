Todays Nearest Asteroids

This app accesses the Near Earth Object Web Service API to retrieve data about the nearest asteroids from Earth today.

The names, miss distance from Earth, velocity and size of the asteroids can be viewed and compared by clicking the < and > arrows. Data is formatted in kilometers. Tap/hover over each sphere to see the corresponding data.
